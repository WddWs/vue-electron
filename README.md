# vue-eletron
------
#vue-eletron有什么功能？


> * 将vue和electron结合
> * 一键开启代码热更新
> * 一键打包
> * 享受vue的开发方式

#命令
> * npm run serve 调试模式
> * npm run build 打包

#关于serve
serve命令监听的地址在main.js中修改
